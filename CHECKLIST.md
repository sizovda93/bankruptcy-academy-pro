## 🆘 Если Что-то Не Работает

1. **Ошибка "Supabase credentials are missing"**
   - Проверь `.env.local` существует и заполнен
   - Перезагрузи dev сервер: `Ctrl+C` → `npm run dev`

2. **Файл не загружается**
   - Проверь консоль браузера: `F12` → Console
   - Убедись bucket "media" публичный
   - Проверь размер файла (макс 10MB)

3. **Таблицы не видны в Supabase**
   - Выполни SQL скрипты еще раз
   - Обновись в браузере: `F5`

4. **Админ панель не открывается**
   - Проверь URL: `http://localhost:5173/admin`
   - Проверь что dev сервер запущен
   - Проверь консоль браузера для ошибок

5. **Пароль не работает**
   - Проверь konsole браузера (F12)
   - Убедись что ввел правильный пароль
   - По умолчанию: `123456`
