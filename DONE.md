| Таблица | Назначение | Поля |
|---------|-----------|------|
| `users` | Пользователи | id, email, full_name, phone, created_at |

## 🚀 Быстрый Старт (10 Минут)

### 1. Создай Supabase проект (2 мин)
```
https://supabase.com → Create project
```

### 2. Выполни SQL скрипты (2 мин)
```
SQL Editor → New Query → Скопируй из SQL-FOR-SUPABASE.sql → Run
```

### 3. Создай bucket (1 мин)
```
Storage → Create bucket → Имя: media → Make public ✓
```

### 4. Заполни .env.local (1 мин)
```env
VITE_SUPABASE_URL=https://твой-проект.supabase.co
VITE_SUPABASE_ANON_KEY=твой-ключ
```

### 5. Запусти приложение (1 мин)
```bash
npm install
npm run dev
```

### 6. Перейди на админ панель (1 мин)
```
http://localhost:5173/admin
Пароль: 123456
```

## 💡 Примеры Использования На Сайте

### Загрузить курсы:
```tsx
const { data: courses } = await supabase
  .from('courses')
  .select('*');
```

### Загрузить отзывы:
```tsx
const { data: reviews } = await supabase
  .from('reviews')
  .select('*')
  .eq('is_published', true);
```

**Все примеры в файле: `SUPABASE-EXAMPLES.tsx`**
