# 🎯 Админ Панель Bankruptcy Academy

Полнофункциональная админ панель с интеграцией Supabase для управления:
- 📚 Курсами и обложками
- 🖼️ Медиа файлами (картинки)
- ⭐ Отзывами от пользователей
- 👥 Пользователями

---

## ⚡ Быстрый Старт

### 1️⃣ Установи зависимости
```bash
npm install
```

### 2️⃣ Создай аккаунт Supabase
Перейди на [https://supabase.com](https://supabase.com) и создай проект

### 3️⃣ Скопируй SQL скрипты
**Файл**: `SQL-FOR-SUPABASE.sql` или `sql-scripts.sql`

В Supabase:
- Открой projekt → SQL Editor → New Query
- Скопируй и выполни весь SQL код

### 4️⃣ Настрой Storage
В Supabase:
- Storage → Create bucket
- Имя: `media`
- Сделай публичным ✓

### 5️⃣ Получи ключи API
Supabase → Settings → API
- Скопируй `Project URL`
- Скопируй `anon public key`

### 6️⃣ Создай `.env.local`
```env
VITE_SUPABASE_URL=https://xxx.supabase.co
VITE_SUPABASE_ANON_KEY=твой-ключ
```

### 7️⃣ Запусти приложение
```bash
npm run dev
```

### 8️⃣ Перейди в админ панель
**URL**: `http://localhost:5173/admin`
**Пароль**: `123456` (измени в коде!)

## 💡 Основное объяснение

### Как загружать картинки?

1. Перейди на вкладку **🖼️ Медиа**
2. Нажми на область или выбери файл (`PNG`, `JPG`, `WebP`)
3. После загрузки нажми **копировать** (📋)
4. Вставь URL где нужно (обложка курса, аватар отзыва и т.д.)

### Как добавить курс?

1. Перейди на вкладку **📚 Курсы**
2. Нажми **Добавить курс**
3. Заполни форму:
   - Название
   - Описание
   - **URL обложки** (скопирована из Медиа)
   - Цена
   - Длительность
   - Уровень
4. Нажми **Сохранить**

### Как добавить отзыв?

1. Перейди на вкладку **⭐ Отзывы**
2. Нажми **Добавить отзыв**
3. Заполни:
   - Имя автора
   - Оценка (1-5 ⭐)
   - Текст отзыва
   - Аватар автора (опционально)
4. Нажми **Опубликовать сразу** если показать на сайте
5. Сохрани

## 📱 Интеграция с основным сайтом

Чтобы показывать данные на сайте, используй:

```tsx
import { supabase } from '@/lib/supabase';

// Загрузить курсы
const { data: courses } = await supabase
  .from('courses')
  .select('*')
  .order('created_at', { ascending: false });

// Загрузить отзывы
const { data: reviews } = await supabase
  .from('reviews')
  .select('*')
  .eq('is_published', true);
```

## ⚠️ Важные моменты

1. **Не коммитьте `.env.local`** в Git (уже в `.gitignore`)

## 🆘 Частые ошибки

| Ошибка | Решение |
|--------|---------|
| "Supabase credentials are missing" | Проверь `.env.local`, перезагрузи сервер |
| Файл не загружается | Проверь размер файла и формат |
| Таблицы не видны | Выполни SQL скрипты в Supabase |
| Не могу копировать URL | Проверь что bucket "media" публичный |
