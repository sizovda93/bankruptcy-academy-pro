# Руководство по настройке Админ Панели и Supabase

## 📋 Шаг 1: Создание проекта Supabase

1. Перейди на [https://supabase.com](https://supabase.com)
2. Нажми "Sign In" или "Sign Up"
3. Создай новый проект:
   - Нажми "New Project"
   - Введи имя проекта
   - Выбери регион (рекомендуется European Union)
   - Установи пароль для администратора БД
   - Нажми "Create new project" и жди загрузки (2-3 минуты)

## 📋 Шаг 2: Выполнение SQL скриптов

1. Перейди в свой проект Supabase
2. Нажми на "SQL Editor" в левом меню
3. Нажми "New Query"
4. Скопируй весь код из файла `sql-scripts.sql` (находится в корне проекта)
5. Вставь в SQL редактор
6. Нажми кнопку "Run" (треугольник справа)

✅ Таблицы должны быть созданы!

## 📋 Шаг 3: Настройка Supabase Storage

1. В левом меню выбери "Storage"
2. Нажми "Create a new bucket"
3. Введи имя: `media`
4. **Неснимай** галочку "Make it public" (нужна для доступа к файлам)
5. Нажми "Create bucket"

✅ Хранилище файлов создано!

## 📋 Шаг 4: Получение API ключей

1. Нажми на иконку ⚙️ (Settings) в левом меню
2. Выбери "API"
3. Скопируй:
   - **Project URL** (выглядит как `https://xxx.supabase.co`)
   - **anon public** (длинный ключ)

## 📋 Шаг 5: Настройка переменных окружения

1. В корне проекта создай файл `.env.local`
2. Скопируй из `.env.example` и заполни:

```env
VITE_SUPABASE_URL=https://твой-проект.supabase.co
VITE_SUPABASE_ANON_KEY=твой-длинный-ключ-здесь
```

⚠️ **ВАЖНО**: Не коммитьте этот файл в Git! Он уже в `.gitignore`

## 📋 Шаг 6: Запуск приложения

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

## 🚀 Использование Админ Панели

**Ссылка**: `http://localhost:5173/admin`

### Команды в админ панели:

**📚 Курсы**
- Добавь новый курс
- Загрузи обложку через "Медиа"
- Скопируй URL обложки и вставь в форму курса

**🖼️ Медиа**
- Нажми на область загрузки или выбери файл
- После загрузки нажми кнопку копирования (📋) рядом с файлом
- Используй URL в других местах сайта

**⭐ Отзывы**
- Добавь автора отзыва
- Set рейтинг (1-5 звёзд)
- Напиши текст отзыва
- Выбери "Опубликовать сразу" если хочешь показать на сайте

**👥 Пользователи**
- Добавляются автоматически при регистрации
- Ты можешь добавить вручную если нужно

## 🔐 Безопасность

В данный момент админ панель **открыта для всех**. Для защиты добавь:

1. **Простая защита с паролем** (быстро):
   - Создай переменную `VITE_ADMIN_PASSWORD` в `.env.local`
   - Добавь check на странице `/admin`

2. **Аутентификация Supabase** (рекомендуется):
   - Используй встроенную систему аутентификации Supabase
   - Требует больше кода, но более безопасно

## 📝 Структура Таблиц БД

```
users (пользователи)
├── id (UUID)
├── email (уникальный)
├── full_name
├── phone
└── created_at

media (загруженные файлы)
├── id (UUID)
├── file_name
├── file_url
├── file_type
├── file_size
└── uploaded_by (ссылка на users)

courses (курсы)
├── id (UUID)
├── title
├── description
├── cover_image_url
├── price
└── level

reviews (отзывы)
├── id (UUID)
├── author_name
├── rating (1-5)
├── comment
├── author_avatar_url
├── is_published
└── created_at

course_registrations (регистрации)
├── user_id (ссылка на users)
└── course_id (ссылка на courses)
```

## 🆘 Решение проблем

**Ошибка: "Supabase credentials are missing"**
- Проверь что `.env.local` создан и заполнен правильно
- Перезагрузи dev сервер: `Ctrl+C` и `npm run dev`

**Ошибка при загрузке файла**
- Проверь что bucket "media" создан и публичный
- Проверь размер файла (макс 10MB)

**Таблицы не видны в Supabase**
- Перейди в SQL Editor и проверь что все скрипты выполнены
- Обновись страницу браузера

## 💾 Интеграция с основным сайтом

Чтобы показывать данные из БД на основном сайте:

```tsx
import { supabase } from '@/lib/supabase';

// Загрузить все курсы
const { data: courses } = await supabase.from('courses').select('*');

// Загрузить опубликованные отзывы
const { data: reviews } = await supabase
  .from('reviews')
  .select('*')
  .eq('is_published', true);
```
